# 🚖 Taxi Revenue Optimization using A/B Testing

## 📌 Overview
This project analyzes taxi trip data to understand how different payment methods (cash vs credit card) impact total fare amount. The goal is to identify data-driven strategies to maximize driver revenue.

---

## 🎯 Objective
- Perform A/B testing to compare fare amounts between payment types  
- Use statistical hypothesis testing to validate differences  
- Generate actionable insights for revenue optimization  

---

## ❓ Research Question
Does payment type influence total fare amount, and can encouraging specific payment methods increase driver revenue?

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib / Seaborn  
- SciPy (Hypothesis Testing)  

---

## 📊 Methodology
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Hypothesis Testing (T-test)  
4. Statistical Interpretation  

---

## 📈 Key Insights
- Credit card payments showed higher average fare compared to cash  
- Statistical testing confirmed a significant difference between groups  
- Suggests potential revenue optimization by promoting digital payments  

---

## 🚀 Business Impact
- Helps taxi platforms optimize pricing strategies  
- Enables drivers to maximize earnings  
- Supports data-driven decision-making
